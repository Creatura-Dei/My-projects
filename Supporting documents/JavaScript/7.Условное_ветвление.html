Когда нужно выполнить действия в зависимости от условий мы используем инструкцию if и условный оператор ? (или же вопросительный знак) 1) Инструкция if if (...) вычисляет условие в скобках и, если результат true - выполняет блок кода. let year = prompt('В каком году была опубликована спецификация ECMAScript-2015?', ''); if (year == 2015) alert( 'Вы правы!' ); Если мы хотим выполнить более одной инструкции, то нужно заключить блок кода в фигурные скобки (лучше в принципе после if всегда ставить их для удобочитаемости): if (year == 2015) { alert( 'Правильно!' ); alert( 'Вы такой умный!' ); } Преобразование к логическому типу: Инструкция if (...) вычисляет выражение в скобках и преобразует результат к логическому типу. Как мы помним: Число 0, пустая строка "", null, undefined, NaN - становятся false. (поэтому их называют ложными значениями) Остальные становятся true - правдивыми То есть, код при таком условии никогда не выполнится: if (0) { ... } //0 if falsy А при таком выполнится: if (1) { ... } //1 is truthy Также можно присвоить переменной само условие if: let condition = (year == 2015); if (condition) { ... } _____________________________________________________ Блок else
