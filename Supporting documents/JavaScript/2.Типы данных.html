Существует 8 типов данных. 1) Число (number) - целочисленные, с плавающей точкой. Существуют операции: умножение *, деление /, сложение +, вычитание - и тд. Помимо обычных чисел, существуют "специальные числовые значения": Infinity, -Infinity, NaN. ________Infinity - мат бесконечность, больше любого числа Получается в результате деления на ноль. alert( 1 / 0 ); ИЛИ alert( Infinity ); ________NaN означает вычислительную ошибку. Это результат неправильной мат операции. alert( "не число" / 2 ); NaN прилипчиво, любая операция с ним возвращает NaN: alert( NaN + 1 ); alert( 3 * NaN); alert( "не число" / 2 - 1); КРОМЕ NaN ** 0 равно 1. ______________________________________________________ 2) BigInt Для нереально огромных чисел, обозначается n в конце числа (например 40 цифр n) const bigInt = 1234567890123456789012345678901234567890n; ______________________________________________________ 3) Строка (sting), должна быть заключена в кавычки let str = "Привет"; let str = 'Одинарные тоже подойдут' let str = `обратные кавычки позволяют встраивать переменные ${str}`; Если одинарные и двойные не имеют различий, то обратные встраивают выражения в строку, заключая в ${...}, например: let name = "Иван"; ____Вставим переменную: alert( `Привет, ${name}!` ); ____Вставим выражение: alert( `результат: ${1 + 2}` ); ______________________________________________________ 4) Булевый (логический boolean) тип Булевый тип может принимать только два значения: true/false. Например: let nameFieldChecked = true; //да, поле отмечено. let ageFieldChechecked = false; //нет, поле не отмечено. ТАКЖЕ могут быть результатом сравнений: let isGreater = 4 > 1; alert( isGreater ); //true (результатом сравнения будет да) ______________________________________________________ 5) null age = null; Специальное значение, которое представляет собой "ничего", "пусто", "значение неизвестно" ______________________________________________________ 6) undefined Это значение формирует тип из самого себя так же, как и null. Оно означает, что "значение не было присвоено" let age; alert(age); // выведет "undefined" ______________________________________________________ 7) Объекты и символы Тип object - особенный. Все остальные типы называются "примитивными", потому что их значениями могут быть только простые значения (будь то строка, или число, или что-то ещё). В объектах же хранят коллекции данных или более сложные структуры. Тип symbol (символ) используется для создания уникальных идентификаторов в объектах. ______________________________________________________ 8) Оператор typeof Возвращает тип аргумента. Это полезно, когда мы хотим обрабатывать значения различных типов по-разному или просто хотим сделать проверку. У него есть две синтаксические формы: //ОБЫЧНЫЙ СИНТАКСИС typeof 5 // Выведет "number" //СИНТАКСИС, напоминающий вызов функции typeof(5) // Также выведет "number" typeof 50 + " Квартир"; // Выведет "number Квартир" typeof (50 + " Квартир"); // Выведет "string" Другими словами, скобки необходимы для определения типа значения, которое получилось в результате выполнения выражения в них typeof undefined // "undefined" typeof 0 // "number" typeof 10n // "bigint" typeof true // "boolean" typeof "foo" // "string" typeof Symbol("id") // "symbol" typeof Math // "object" (1) typeof null // "object" (2) typeof alert // "function" (3)
