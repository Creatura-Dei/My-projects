1) alert - функция окна, ждёт, чтоб нажали назад. Называется модальным окном. (юзер не может взаимодействовать с сайтом пока не подтвердит) 2) prompt - функция принимает два аргумента: result = prompt(title, [default]); где title - текст для отображения в окне default - необязательный второй параметр, который вставит изначально в поле ввода, выделяя это * Квадратные скобки [] означают, что параметр факультативный, необязательный. Например: let age = prompt("Сколько тебе лет?", 100); alert(`Тебе ${age} лет!`); Для IE всегда указывается второй параметр: let test = prompt("Test", ''); 3) Confirm - (окно выбора кароч) функция отображает модальное окно с текстом вопроса и двумя кнопками ОК и Отмена. true - если нажата ОК false - во всех других случаях let isBoss = confirm("Ты здесь главный?"); alert( isBoss );
