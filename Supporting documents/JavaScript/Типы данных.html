Существует 8 типов данных. 1) Число (number) - целочисленные, с плавающей точкой. Существуют операции: умножение *, деление /, сложение +, вычитание - и тд. Помимо обычных чисел, существуют "специальные числовые значения": Infinity, -Infinity, NaN. ________Infinity - мат бесконечность, больше любого числа Получается в результате деления на ноль. alert( 1 / 0 ); ИЛИ alert( Infinity ); ________NaN означает вычислительную ошибку. Это результат неправильной мат операции. alert( "не число" / 2 ); NaN прилипчиво, любая операция с ним возвращает NaN: alert( NaN + 1 ); alert( 3 * NaN); alert( "не число" / 2 - 1); КРОМЕ NaN ** 0 равно 1.
