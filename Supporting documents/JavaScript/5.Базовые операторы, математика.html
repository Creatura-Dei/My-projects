Оператор - команда, обозначающая опр математическое действие, выполняемое с данными - операндами. (например + * -) УНАРНЫЙ, БИНАРНЫЙ, ОПЕРАНД Операнд - то, к чему применяется оператор. Например, в умножении 5 * 2 есть два операнда: левый = 5, а правый = 2. Иногда их называют аргументами вместо операндов. Унарный - оператор, который применяется к одному операнду. Например, - меняет знак числа на противоположный. let x = 1; x = -x; alert( x ); // -1 Бинарный - оператор, который применяется к двум операндам. let x = 1, y = 3; alert( y - x ); // 2 ______________________________________________________________________ Мне доступны: Сложение + Вычитание - Умножение * Деление / Взятие остатка от деления % Возведение в степень ** ______________________________________________________________________ 1) Взятие остатка % Оператор никакого отношения к процентам не имеет. Результат a % b - это остаток от цисленного деления a на b. Например: alert( 5 % 2 ); // 1, остаток от деления 5 на 2 alert( 8 % 3 ); // 2 alert( 8 % 4 ); // 0 2) Возведение в степень ** Оператор возведения a ** b возводит a в степень b. Например: alert( 2 ** 2 ); // 4 alert( 2 ** 3 ); // 8 alert( 2 ** 4 ); // 16 + оператор работает и для нецелых чисел. Например: alert( 4 ** (1/2) ); // 2 alert( 8 ** (1/3) ); // 2 ______________________________________________________________________ 1) Сложение строк при помощи бинарного + Если бинарный оператор + применить к строкам, то он их объединяет в одну: let s = "моя" + "строка"; alert(s); // моястрока ЕСЛИ хотя бы один операнд является строкой, то второй также будет преобразован в строку. alert( '1' + 2 ); // 12 alert( 2 + '1' ); // 21 alert( 2 + 2 + '1' ); // будет 41, а не 221 Здесь операторы работают один за другим. !!!!!!!!!!!!!!!!! Такая тема работает только с бинарным плюсом. !!!!!!!!!!!!!!!!! 2) Приведение к числу, унарный + Если операнд не число, унарный плюс преобразует его в число. Например: // Не влияет на числа let x = 1; alert( +x ); // 1 let y = -2; alert( +y ); // -2 // Преобразует не числа в числа alert( +true ); // 1 alert( +"" ); // 0 На самом деле это то же самое, что и Number(...), только короче. Часто приходится значения полей HTML-формы - строки - сложить. НЕПРАВИЛЬНО let apples = "2"; let oranges = "3"; alert( apples + oranges ); //23, тк бинарный плюс складывает строки ПРАВИЛЬНО let apples = "2"; let oranges = "3"; alert( +apples + +oranges ); //5 ИЛИ ЖЕ alert( Number(apples) + Number(oranges) ); //5 ПРИОРИТЕТ УНАРНЫХ НАД БИНАРНЫМИ ______________________________________________________ ПРИСВАИВАНИЕ 1) Присваивание = возвращает значение let a = 1; let b = 2; let c = 3 - (a = b + 1); alert( a ); // 3 alert( c ); // 0 2) Присваивание по цепочке - РАБОТАЕТ СПРАВА НАЛЕВО let a, b, c; a = b = c = 2 + 2; alert( a ); alert( b ); alert( c ); //везде будет 4 ТО ЕСТЬ: сначала 2+2, потом присваивается c, b, a. 3) Сокращённая арифметика с присваиванием (имеет такой же приоритет, что и обычное присваивание, обрабатывается почти в самом конце) Часто нужно применить оператор к переменной и сохранить результат в ней же Например: let n = 2; n = n + 5; n = n * 2; НО МОЖНО использовать сокращённые операторы: += и *= и /= и -= let n = n += 5; // теперь 7 n *= 2; // теперь 14 alert( n ); // 14 ______________________________________________________ Инкремент и декремент - плюс один и минус один let counter = 2; counter++; или же counter--; alert( counter ); ВАЖНО! Они применяются ТОЛЬКО к ПЕРЕМЕННОЙ. Если сделаешь 5++ - ничего не получится !!! Если результат оператора не используется, а нужно только увеличить/уменьшить переменную, тогда без разницы, какую форму использовать: let counter = 0; counter++; ++counter; alert( counter ); // 2, обе строки сделали одно и то же !!! Если хочется тут же использовать результат, то нужна префиксная форма: let counter = 0; alert( ++counter ); // 1 !!! Если нужно увеличить и при этом получить значение переменной до увеличения – нужна постфиксная форма: let counter = 0; alert( counter++ ); // 0 ______________________________________________________________________ Побитовые операторы Поддерживаются следующие побитовые операторы: AND(и) ( & ) OR(или) ( | ) XOR(побитовое исключающее или) ( ^ ) NOT(не) ( ~ ) LEFT SHIFT(левый сдвиг) ( << ) RIGHT SHIFT(правый сдвиг) ( >> ) ZERO-FILL RIGHT SHIFT(правый сдвиг с заполнением нулями) ( >>> ) ______________________________________________________________________ Оператор "запятая" ( , ) - применяется редко. Предоставляет возможность вычислять несколько выражений, разделяя их запятой,. Каждое выражение выполняется, но возвращается результат только последнего. let a = (1 + 2, 3 + 4); alert( a ); // 7 (результат вычисления 3 + 4) Запятая имеет очень низкий приоритет
